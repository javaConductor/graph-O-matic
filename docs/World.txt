GetItemViewInfo = function(itemType, item) { ..., return viewInfo;}
_$itypes$<TheItemTypeName>$fViewInfo    -   Called to get the viewInfo related to an item with itemType named 'TheItemTypeName'.
_$itypes$fViewInfo                      -   The default function for getting the viewInfo related to an item.
